Metro 是一个 Minecraft 地铁交通系统插件，让你在服务器中创建自动化的地铁网络。玩家右键点击红石铁轨即可呼叫矿车并自动乘坐。

## ✨ 特性

- **多线路网络** — 创建多条线路，设置停靠区和换乘
- **图形化管理** — 内置 GUI 界面，无需记忆命令
- **定价系统** — 支持单一票价、固定票价、按距离/站数计价
- **权限管理** — 元素级信任与所有权系统
- **安全模式** — 保护矿车免受推动、攻击和破坏
- **矿车传送门** — 跨区域 / 跨世界传送
- **网页地图** — 支持 BlueMap / Dynmap / Squaremap
- **经济系统** — 可选 Vault 集成
- **多语言** — 支持中英文等多国语言
- **Folia 兼容** — 支持 Folia 多线程服务端

## 📖 基本概念

| 概念 | 说明 |
| :--- | :--- |
| **线路 (Line)** | 按顺序排列的停靠区列表，定义列车行驶路径 |
| **停靠区 (Stop)** | 由两个对角点定义的三维站台区域 |
| **停靠点 (StopPoint)** | 停靠区内的红石铁轨，玩家右键乘车的位置 |
| **换乘 (Transfer)** | 在停靠区转乘其他线路 |
