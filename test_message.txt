Instead, the correct way to get the world in this version is through `api.getWorldIfEnabled(xyz.jpenilla.squaremap.api.BukkitAdapter.worldIdentifier(bukkitWorld))`.

I have updated `SquaremapIntegration.java` to use the correct API method and confirmed that `mvn clean package` now successfully compiles the plugin without any errors.
